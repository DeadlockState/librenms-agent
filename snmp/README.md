# /usr/bin/distro

## Description 

Added Proxmox VE support on distro script for Observium and LibreNMS

## Installation

```shell
cd /usr/bin/

wget https://raw.githubusercontent.com/Punk--Rock/librenms-agent/master/snmp/distro

chmod +x distro
```
